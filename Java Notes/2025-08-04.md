## How to change String '2' to Integer 2

```java
int digit = num.charAt(i) - '0;
```
- **`num.charAt(i)` gives you a character, like `'2'`.**
    
- **In Java, characters are stored as ASCII/Unicode values.**
    
    - **`'0'` has a value of 48**
        
    - **`'1'` is 49**
        
    - **`'2'` is 50, and so on.**
        
- **So `'2' - '0'` becomes:**  
    **`50 - 48 = 2` âœ…**
    
**This works for all digits `'0'` to `'9'`**
